# Shell-Script

1️⃣ Docker Install

```
sudo su - 
mkdir git
cd git
git clone https://github.com/ohy1023/Shell-Script.git
cd Shell-Script
sh docker_install.sh
```

2️⃣ docker-compose install

```
sudo su - 
mkdir git
cd git
git clone https://github.com/ohy1023/Shell-Script.git
cd Shell-Script
sh docker_compose_install.sh
```

3️⃣ MYSQL(MASTER-SLAVE) Run

```
sudo su - 
mkdir git
cd git
git clone https://github.com/ohy1023/Shell-Script.git
cd Shell-Script
cd shopping-mall-project
docker-compose up -d
```
